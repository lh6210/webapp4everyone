array.sort() with numbers

Nov 4, 2020

In this article, let's look at array.sort() method.

sort() method applies on JS's arrays and by default it first converts every array element into a string and then arranges them according to their string values. So if we have an array of numbers, sort() seems weird.


	  let arr = new Array(3, 2, 21, 15, 10);
	  arr.sort();
	  

	  console.log(arr);
	  (out)🠞 Array(5) [ 10, 15, 2, 21, 3]

But this is because sort() compares them in string values and the result is a correct ascending order in that sense: 


	  "10" < "15" && "15" < "2" && "2" < "21" && "21" < "3";
	  (out)🠞 True

Now we know that it's just sort() method's default logic. And we could implement our sorting logic onto it, too.


	  function compareInNumeric(v1, v2) {
	      return v1 - v2;
	  }
	  arr.sort(compareInNumeric);
	  

	  console.log(arr);
	  (out)🠞 array(5) [2, 3, 10, 15, 21]

As shown above, compareInNumeric makes the sorting as expected.
By the way, sort() method could take in one argument, which is the name or pointer of the comparison method (compareInNumeric in this case) evaluating the order among elements. The comparison method takes in two arguments (v1, v2 in our example), and shows its pereference of one over the other for a good reason. The comparison method that we are using above doesn't transform elements into strings. Instead, it assumes those arguments are numbers and directly compare them with their numeric values.
